Tp réalisé avec :

- InteliJ
- JDK 11
- Tomcat 9.0.19
- mySQL
- Sping Boot

Dans le répertoire DB vous trouverez la commande pour créer la DB

Dans src/main/resources vous trouverez le fichier application.properties.
    Ici, vous devez renseigner les champs username et password

Dans le répertoire Front vous trouverez les pages html
